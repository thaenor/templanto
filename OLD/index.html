Translation engine

working title

This translation engine uses mustache render engine to generate several HTML files based on a template.
It relies on a specific folder structure, as it reads from that and outputs the files to the /render folder.

Before we begin

  • The excel files with translations must be in the “translation_source” folder and follow the format (case sensitive).
     
| key   | value |
    | HELLO | olá   |
    | BODY  | texto |

 

  • this project was tested on the LTS version of node (V.6.11.0, npm v.4.6.1) - it should run on newer versions though.

How to use

  • Inside the “templates” folder, create a folder with the same name as the excel sheet.
  • Inside that folder create your template
    • the template must be a .Mustache file - these are normal files with {{ this-kind-of-notation }}
    • the keys must be the same referenced in your excel spreadsheet
  • open terminal and go to the project folder
  • run npm start

the result should look something like this:
 

./templates
    ---- /my-page-template
    ---- ---- /translations
    ---- ---- ---- /EN.JSON
    ---- ---- ---- /FR.JSON
    ---- ---- ---- /DE.JSON (... you should have as many JSON files as the sheets in the excel)
    ---- ---- my-page-template.mustache
    /render
    ---- /my-page-template
    ---- ---- my-page-template-EN.html
    ---- ---- my-page-template-FR.html
    ---- ---- my-page-template-DE.html

 

example of a template file

<!DOCTYPE HTML>
        <html>
        <body>
            <h1>Hello {{name}}!</h1>
        </body>
    </html>

in this example {{name}} will be replaced with the translation.

use {{{name}}} if you want to keep literal code in the translation.
